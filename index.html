AutoAnalyst: Autonomous AI Data Analysis Agent

Personal project

Drop in a CSV and get back a full report with charts, stats, and recommendations. Five LangGraph agents run in sequence: Profiler, Planner, Analyst, Critic, and Reporter. If the Critic scores the analysis below 7/10, the Analyst goes back for another pass. Progress streams live to the React/TypeScript frontend via SSE, and once the report is done, everything is stored in ChromaDB so you can ask follow-up questions.
